# 第一部分：渐近理论根基 — 全局逻辑出发点

> 三层地基：渐近序列（合法性）→ 渐近展开（余项控制）→ 超越小量（全局源头）
> 核心逻辑链：指数放大 → 压缩区间 → 峰值主导 → 泰勒近似 → 高斯积分终结
>
> ### 🔥 考试重点（老师强调）
> **Laplace 标准操作流程**（老师口述）：① 先缩积分上限（确定 $\phi(t)$ 最大值点 $t_c$，压缩到 $t_c$ 附近小邻域）→ ② 延展积分限到 $\pm\infty$（尾部指数级小，可忽略）→ ③ 驻点局部 Taylor 展开 → ④ 高斯积分终止。
> **Laplace + 稳定相 = 两步走方法**：① 确定驻点 $\phi'(t_c)=0$ → ② 驻点局部展开 + 高斯积分终止。两个方法共享同一套技术链，区别仅在于实指数（尖峰）vs 虚指数（振荡抵消）。无驻点情形用**分部积分法**处理。经典例子：Gamma → Stirling，Bessel $J_0(x)$ 大 $x$ 渐近。考试中需要的积分公式会给出。
> ### ⭐ 要求掌握
> 渐近序列定义 · 渐近展开 Poincaré 定义与余项控制 · **非标准序列展开**（给定 $\{\phi_n\}$ 按定义逐项求系数，如 $\sqrt{25+x}$ 展为 $\{1,\cos x,\sin x,\sin^2 x\}$）· 超越小量 $e^{-1/\epsilon}$ 的特性与后果

<a id="sec-1"></a>
## 1. 拉普拉斯方法 (Laplace's Method) 🔥

### 1.1 问题形式

考虑积分（$x \to \infty$）：

$$I(x) = \int_a^b f(t)\, e^{x\phi(t)} dt$$

### 1.2 核心直觉

当 $x \to \infty$，$e^{x\phi(t)}$ 在 $\phi(t)$ 取**最大值**的点 $t_c \in (a,b)$ 处形成一个**无限尖锐的峰**。峰值宽度 $\sim 1/\sqrt{x}$，只有 $t_c$ 附近的小邻域对积分有贡献。尾部（远离 $t_c$ 的区域）的贡献**指数级小**，可以忽略。

### 1.3 例 1：高斯积分（粗糙估计）

$$I = \int_{-\infty}^{\infty} e^{-xt^2} dt$$

- 被积函数在 $t=0$ 有尖锐峰，宽度 $\sim 1/\sqrt{x}$，高度 $=1$
- 粗糙估计：$I \sim O(1/\sqrt{x})$
- **精确值**：$I = \sqrt{\pi/x}$
- **尾部验证**：$\int_{10}^{\infty} e^{-xt^2} dt < \int_{10}^{\infty} e^{-10xt} dt = \frac{e^{-100x}}{10x} \ll O(1/\sqrt{x})$，可忽略

### 1.4 例 2：$\int_{-\infty}^{\infty} e^{-x\cosh t} dt$

- $\cosh t = (e^t + e^{-t})/2$，在 $t=0$ 取最小值 $1$（对应 $-\cosh t$ 取最大值 $-1$）
- 在 $t=0$ 展开：$\cosh t = 1 + t^2/2! + t^4/4! + \cdots$
- **仅保留到 $t^2$** 即可（高阶项 $\times$ 指数衰减 → 可忽略）
- 近似：$I(x) \approx \int_{-\infty}^{\infty} e^{-x(1 + t^2/2)} dt = \sqrt{\frac{2\pi}{x}} e^{-x}$
- 若需高阶修正，通过换元 $\cosh t = 1 + 2\sigma^2$，将积分化为高斯积分族，得到**完整渐近级数**：
  $$I(x) \sim \sqrt{\frac{2\pi}{x}} e^{-x} \left(1 - \frac{1}{8x} + \frac{9}{128x^2} + \cdots\right)$$

> **关键注**：无论 $O(\sigma^4)$ 项"多错"，它们都乘以指数级小的 $e^{-x\sigma^2}$。

### 1.5 例 3：Gamma 函数与 Stirling 公式

**Gamma 函数定义**：

$$\Gamma(x+1) = \int_0^{\infty} t^x e^{-t} dt$$

- 对整数 $n$：$\Gamma(n+1) = n!$
- 无简单封闭形式——是"积分表示的函数"典型代表
- 其他例子：误差函数 $\text{erf}(x) = \frac{2}{\sqrt{\pi}}\int_0^x e^{-t^2}dt$、Bessel 函数 $J_0(x) = \frac{1}{\pi}\int_0^\pi \cos(x\sin t)dt$

**用拉普拉斯方法求渐近**：

1. 改写为拉普拉斯形式：$\Gamma(x+1) = \int_0^{\infty} e^{x\ln t - t} dt$（$\phi(t) = \ln t - t/x$，但更自然地写为 $\int_0^{\infty} e^{x\ln t - t} dt$）
2. 对指数求导找临界点：$\frac{d}{dt}(x\ln t - t) = \frac{x}{t} - 1 = 0 \Rightarrow t_c = x$
3. 换元 $t = x\tau$，$dt = x d\tau$：$\Gamma(x+1) = x^{x+1} e^{-x} \int_0^{\infty} e^{x(\ln\tau - \tau)} d\tau$
4. 在 $\tau=1$ 展开 $\ln\tau - \tau = -1 - \frac{(\tau-1)^2}{2} + \frac{(\tau-1)^3}{3} - \cdots$
5. 保留到高斯项：$\Gamma(x+1) \sim x^{x+1} e^{-x} \sqrt{\frac{2\pi}{x}}$

**Stirling 公式**：

$$n! \sim n^n e^{-n} \sqrt{2\pi n}, \qquad \ln n! = n\ln n - n + O(\ln n)$$

## 2. 渐近展开 (Asymptotic Expansion) 🔥

### 2.1 三个核心记号

| 记号 | 含义 | 定义 |
|------|------|------|
| $f \ll g$（小o / 远小于） | $f$ 比 $g$ 衰减快得多 | $\lim_{x\to x_0} f/g = 0$ |
| $f \sim g$（渐近等价） | $f$ 与 $g$ 同阶 | $\lim_{x\to x_0} f/g = 1$ |
| $f = O(g)$（大O） | $f$ 的量级不超过 $g$ | $\limsup_{x\to x_0} \|f/g\| < \infty$ |

**例**：$\sin x - x = O(x^3)$ as $x \to 0$

### 2.2 关键辨析

- $\sim$ **不蕴含**数值近似：$e^x + x \sim e^x$（$x\to\infty$），但数值差异巨大
- $f \ll g$ 也不蕴含 $\sim$：$x^2 \ll x^3$ near $x=0$，但它们不渐近等价
- 非零函数**不能**渐近于零

### 2.3 渐近展开的 Poincaré 定义

$f(x)$ 关于渐近序列 $\{\phi_n(x)\}$（$\phi_{n+1} \ll \phi_n$）有渐近展开：

$$f(x) \sim \sum_{n=1}^{\infty} a_n \phi_n(x) \quad (x \to x_0)$$

当且仅当对每个 $N$：

$$R_N(x) \equiv f(x) - \sum_{n=1}^N a_n \phi_n(x) = O(\phi_{N+1}(x))$$

即**余项远小于最后保留的项**。

### 2.4 例：$I(x) = \int_0^{\infty} \frac{e^{-xt}}{1+t} dt$

这是贯穿始终的核心例子。

**推导**：
1. 几何级数展开：$\frac{1}{1+t} = 1 - t + t^2 - t^3 + \cdots$（$|t|<1$ 时收敛）
2. $I(x) \sim \sum_{n=0}^{\infty} (-1)^n \int_0^{\infty} e^{-xt} t^n dt$
3. 分部积分 $n$ 次：$\int_0^{\infty} e^{-xt} t^n dt = \frac{n!}{x^{n+1}}$
4. 故 $I(x) \sim \sum_{n=0}^{\infty} (-1)^n \frac{n!}{x^{n+1}} = \frac{1}{x} - \frac{1}{x^2} + \frac{2!}{x^3} - \frac{3!}{x^4} + \cdots$

**该级数是发散的**：$|a_{n+1}/a_n| = (n+1)/x \to \infty$。发散源：几何级数 $\frac{1}{1+t}$ 在 $|t|>1$ 时发散。

**数值例**：$x=10$ 时真值 $=0.09156$，部分和给出 $0.0914$，误差仅 $0.00016$。

**与不完全 Gamma 函数的关系**：做代换 $\sigma = (1+t)x$，可得 $I(x) = e^x \Gamma(0, x)$（上不完全 Gamma 函数）。

### 2.5 余项估计与截断准则

对于上述级数的 $N$ 项部分和 $S_N(x)$，余项为：

$$R_N = (-1)^N \int_0^{\infty} e^{-xt} \frac{t^N}{1+t} dt$$

利用 $\frac{1}{1+t} \leq 1$，得：

$$|R_N| < \int_0^{\infty} e^{-xt} t^N dt = \frac{N!}{x^{N+1}} = |a_{N+1}|$$

**余项的绝对值不超过第一个被略去的项的绝对值。**

由于 $R_N$ 符号交替，真值被相邻两个部分和夹逼在中间：

$$S_{N-1} < I(x) < S_N \quad \text{或} \quad S_N < I(x) < S_{N-1}$$

**最优截断**：当两项夹逼最紧时停止。

### 2.6 渐近展开的一般性质

#### (1) 收敛级数 vs 渐近级数

| | 收敛级数 | 渐近级数 |
|---|---|---|
| 极限过程 | 固定 $x$，$N \to \infty$ | 固定 $N$，$x \to x_0$ |
| 关切 | 无限项和是否趋于 $f(x)$ | 有限项是否逼近 $f(x)$ |

**收敛 Taylor 级数是渐近展开的特例。**

#### (2) 系数的唯一性（给定渐近序列）

$$a_1 = \lim_{x\to x_0} \frac{f(x)}{\phi_1(x)}, \quad a_2 = \lim_{x\to x_0} \frac{f(x) - a_1\phi_1(x)}{\phi_2(x)}, \quad \cdots$$

但渐近序列 $\{\phi_n\}$ 的**选择不唯一**。

**非标准序列的例子**（老师点名考点）：$f(x) = \sqrt{25+x}$ 在 $x \to 0$，给定渐近序列 $\{1, \cos x, \sin x, \sin^2 x\}$。

**按定义展开的标准步骤**：
1. $a_1 = \lim_{x\to 0} f(x)/1 = 5$
2. $a_2 = \lim_{x\to 0} \frac{f(x) - 5}{\cos x} = \lim_{x\to 0} \frac{\sqrt{25+x} - 5}{\cos x} = \frac{1/10}{1} = \frac{1}{10}$（分子 Taylor：$5(1 + x/50 - x^2/5000 + \cdots) - 5 = x/10 - x^2/200$）
3. $a_3 = \lim_{x\to 0} \frac{f(x) - 5 - \frac{1}{10}\cos x}{\sin x} = \cdots$
4. 逐次分解，每个系数由极限唯一确定

**关键**：渐近序列 $\{\phi_n\}$ 的**选择不唯一**——同样的 $f(x)$ 对不同渐近序列有不同的 $a_n$。但给定序列后系数唯一。考试要求掌握"按定义展开"的完整步骤，而非死记展开公式。

> 原笔记例子：$f(x) = \sqrt{1+x}$，取 $\{\cos x, \sin x, \sin^3 x\}$。步骤同上。

#### (3) 超越小项

若 $f(x) \ll 1/x^\alpha$ 对**任意** $\alpha > 0$ 成立，则 $f$ 是**超越小**的。

**例**：$e^{-x}$（$x \to \infty$），因为 $\lim_{x\to\infty} e^{-x} / x^{-\alpha} = 0$。

**后果**：$f(x)$ 和 $f(x) + ce^{-x}$ 具有相同的幂次渐近展开——超越小项在幂律渐近级数中"不可见"。

#### (4) 渐近展开的有效/无效操作

| 允许 | 需谨慎/不允许 |
|------|---------------|
| 加减 | 逐项代换 $f(g(x))$ |
| 乘除 | 逐项求导 |
| 逐项积分 | — |

## 3. 稳定相方法 (Method of Stationary Phase) 🔥

### 3.1 基本问题

$$I(x) = \int_a^b f(t)\, e^{ix\psi(t)} dt, \quad x \to \infty$$

### 3.2 Riemann-Lebesgue 引理

若 $f(t)$ 在 $[a,b]$ 上可积，则经典 R-L 引理给出：

$$\lim_{x \to \infty} \int_a^b f(t) e^{ixt} dt = 0 \quad \text{即 } o(1)$$

若进一步 $f \in C^1$（一阶连续可导），通过分部积分可得更强的估计：

$$I(x) = O\left(\frac{1}{x}\right)$$

**直觉**：快速振荡导致正负抵消。

对 $\psi(t) = t$ 的特例：

$$\int_a^b f(t) e^{ixt} dt = O\left(\frac{1}{x}\right)$$

**分部积分证明**：$\int_a^b f(t) e^{ix\psi(t)} dt = \left.\frac{f(t)}{ix\psi'(t)} e^{ix\psi(t)}\right|_a^b - \frac{1}{ix}\int_a^b e^{ix\psi(t)} \frac{d}{dt}\left(\frac{f(t)}{\psi'(t)}\right) dt$

第一项 $O(1/x)$，第二项 $O(1/x^2)$（若 $f/\psi'$ 光滑）。

### 3.3 稳定相点

若 $\psi'(t_s) = 0$（$t_s \in [a,b]$），则振荡在 $t_s$ 附近"减速"，抵消最弱，成为积分的主要贡献来源。$t_s$ 称为**稳定相点**。

### 3.4 例 1：$\int_0^{\infty} \cos(x(t^3/3 - t)) dt$

- $\psi(t) = t^3/3 - t$，$\psi'(t_s) = t_s^2 - 1 = 0 \Rightarrow t_s = 1$
- 在 $t_s=1$ 展开：$\psi(t) = -2/3 + (t-1)^2 + O((t-1)^3)$
- $I(x) \approx \cos(2x/3) \int_{-\infty}^{\infty} \cos(xu^2) du + \sin(2x/3) \int_{-\infty}^{\infty} \sin(xu^2) du$

利用 **Fresnel 积分**：

$$\int_0^{\infty} \sin(xt^2) dt = \sqrt{\frac{\pi}{8x}}, \quad \int_{-\infty}^{\infty} e^{-is^2} ds = \sqrt{\pi} e^{-i\pi/4}$$

最终 $I(x) = O(1/\sqrt{2x})$（比 $O(1/x)$ 大，因为稳定相点的存在减弱了抵消）。

### 3.5 例 2：Bessel 函数 $J_0(x)$ 的渐近

**积分表示**：$J_0(x) = \frac{1}{\pi}\int_0^{\pi} \cos(x\sin t) dt = \frac{1}{\pi}\text{Re}\int_0^{\pi} e^{ix\sin t} dt$

- $\psi(t) = \sin t$，$\psi'(t_s) = \cos t_s = 0 \Rightarrow t_s = \pi/2$
- 展开：$\sin t = 1 - \frac{1}{2}(t - \pi/2)^2 + \cdots$
- 代入得 $J_0(x) \sim \frac{1}{\pi} \text{Re}\left[e^{ix} \int e^{-ix(t-\pi/2)^2/2} dt\right]$

换元 $s = \sqrt{x/2}(t-\pi/2)$，用到 **Fresnel 积分与复路径**：

$$\int_0^{\infty} e^{-is^2} ds = e^{-i\pi/4} \int_0^{\infty} e^{-r^2} dr = \frac{\sqrt{\pi}}{2} e^{-i\pi/4}$$

（沿 $z = re^{-i\pi/4}$ 的射线做围道积分，利用 $e^{-iz^2}$ 的解析性。）

**最终结果**：

$$J_0(x) \sim \sqrt{\frac{2}{\pi x}} \cos\left(x - \frac{\pi}{4}\right), \qquad x \to \infty$$

# 第二部分：通用前置基础 — 从积分到 ODE 的桥梁

> 三件工具：分部积分法 + Rescaling（重缩放）+ Dominant Balance（主导平衡）
> 分部积分是"无驻点情形"的标准对策，也是从积分渐近过渡到边界层分析的**关键桥梁**
>
> ### 🔥 考试重点（老师强调）
> **Rescaling + 主导平衡**是后续所有 ODE 摄动方法的技术根基：
> - 涉及伸缩变换确定边界层厚度 $\delta = \epsilon^\gamma$，$\gamma$ 通过主导平衡确定
> - $y = e^S$ 拟设 → $S'' \ll (S')^2$ → $(S')^2 \sim f(x)$ → 逐阶修正
> - 对非正则奇点的 Airy 方程应用是典型考题

<a id="sec-4"></a>
## 4. ODE 的局部分析 (Local Analysis of ODEs) ⭐

### 4.1 二阶线性齐次 ODE

$$y'' + P(x)y' + Q(x)y = 0$$

通解：$y = c_1 y_1(x) + c_2 y_2(x)$（两个线性无关解的线性组合）

### 4.2 $x_0$ 的分类

根据 $P(x)$ 和 $Q(x)$ 在 $x_0$ 处的解析性：

| 类型 | 条件 | 解法 |
|------|------|------|
| **常点 (Ordinary)** | $P, Q$ 在 $x_0$ 解析 | Taylor 展开 |
| **正则奇点 (Regular Singular)** | $(x-x_0)P(x)$ 和 $(x-x_0)^2 Q(x)$ 在 $x_0$ 解析 | **Frobenius 级数** $y = \sum_{n=0}^{\infty} a_n (x-x_0)^{n+\alpha}$，$\alpha$ 为指标根 |
| **非正则奇点 (Irregular Singular)** | 以上条件都不满足 | 无通用方法，需用主导平衡 |

### 4.3 例 1：$x^n y'' + xy' + y = 0$

化为标准形：$y'' + \frac{1}{x^{n-1}} y' + \frac{1}{x^n} y = 0$，分析 $x_0 = 0$：

- $n=0$：$P(x)=x$（$1/x^{-1}=x$），$Q(x)=1$ → **常点**
- $n=1$：$P(x)=1$，$Q(x)=1/x$ → **正则奇点**
- $n=2$：$P(x)=1/x$，$Q(x)=1/x^2$ → **正则奇点**
- $n \geq 3$：$P(x)=1/x^{n-1}$，$Q(x)=1/x^n$ → **非正则奇点**

### 4.4 例 2：Airy 方程在无穷远处的局部分析

$y'' = xy$，研究 $x \to \infty$。令 $t = 1/x$，则 $x = 1/t$，$dx/dt = -1/t^2$。

$$y' = \frac{dy}{dx} = \frac{dy}{dt}\frac{dt}{dx} = -t^2 \dot{y}$$

$$y'' = \frac{d}{dx}(-t^2 \dot{y}) = (-t^2 \ddot{y} - 2t\dot{y})(-t^2) = t^4 \ddot{y} + 2t^3 \dot{y}$$

代入：$t^4 \ddot{y} + 2t^3 \dot{y} = \frac{1}{t} y$

除以 $t^4$：$\ddot{y} + \frac{2}{t}\dot{y} - \frac{1}{t^5}y = 0$

在 $t_0 = 0$，这是一个**非正则奇点**。对此类奇点，采用的策略是**指数拟设** $y = e^{S(x)}$，即下面要讲的主导平衡方法。

## 5. 主导平衡 (Dominant Balance) 🔥

### 5.1 方法思路

对类似 $y'' = x^{-3} y$（$x \to 0^+$）的 ODE（注意笔记中以 $y'' = x^{-3}y$ 为例），假定解的形式：

$$y(x) = e^{S(x)}$$

求导：
$$y' = S' e^S, \quad y'' = [S'' + (S')^2] e^S$$

代入 ODE：$S'' + (S')^2 - f(x) = 0$（$f(x) = x^{-3}$）

**核心假设**：$S'' \ll (S')^2$（主导平衡），于是：

$$(S')^2 \sim x^{-3} \quad\Rightarrow\quad S' \sim \pm x^{-3/2}$$

积分：$S \sim \mp 2x^{-1/2}$

### 5.2 一般规则

若 $S' \sim 1/x^n$（$x \to 0^+$），积分行为：

| $n$ | $S(x)$ 的行为 |
|-----|--------------|
| $n > 1$ | $S \sim \frac{1}{n-1} x^{-(n-1)}$ |
| $n = 1$ | $S \sim \ln x$ |
| $n < 1$ | $S \sim x^{1-n}$ |

### 5.3 Airy 方程的主导平衡（详细）

（以 $y'' = x^{-3}y$ 为例，$x \to 0^+$。）假设 $S$ 的展开，取负号分支 $S \sim -2x^{-1/2}$。设 $S = -2x^{-1/2} + C(x)$，其中 $C$ 缓慢变化。

逐代修正：
1. $S' = x^{-3/2} + C'$，$S'' = -\frac{3}{2}x^{-5/2} + C''$
2. 代入 $S'' + (S')^2 - x^{-3} = 0$：
   $$-\frac{3}{2}x^{-5/2} + C'' + (x^{-3/2} + C')^2 - x^{-3} = 0$$
3. 展开平方项并保留主导项，得 $C' \sim \frac{3}{4}\frac{1}{x}$，故 $C \sim \frac{3}{4}\ln x$
4. 代入再求下一阶：$D' \sim -\frac{3}{32x}$，$D \sim -\frac{3}{16}x^{-1/2}$

**最终解**：

$$y(x) \sim C e^{\pm 2x^{-1/2}} x^{3/4} \left(1 \mp \frac{3}{16}x^{1/2} + \cdots\right)$$

## 6. 摄动方法—代数方程 (Perturbation Methods)

### 6.1 正则摄动

含小参数 $\epsilon$ 的方程，假设解按 $\epsilon$ 的幂展开：

$$x = x_0 + \epsilon x_1 + \epsilon^2 x_2 + \cdots$$

代入原方程，按 $\epsilon$ 各次幂收集成线性方程。

**核心目标**：将非线性方程转化为一系列线性方程。

**缺陷**：需要预先猜测正确的渐近序列。

### 6.2 例 1：$x^2 + \epsilon x - 1 = 0$

精确解：$x = \frac{-\epsilon \pm \sqrt{\epsilon^2 + 4}}{2} = \pm 1 \mp \frac{\epsilon}{2} \pm \frac{\epsilon^2}{8} + \cdots$

摄动展开（正根，$\epsilon=0$ 时 $x=1$）：
$$x = 1 + \epsilon x_1 + \epsilon^2 x_2 + \cdots$$

代入：
- $O(1)$：$1 - 1 = 0$ ✓
- $O(\epsilon)$：$2x_1 + 1 = 0 \Rightarrow x_1 = -1/2$
- $O(\epsilon^2)$：$x_1^2 + 2x_2 + x_1 = 0 \Rightarrow x_2 = 1/8$

$$x = 1 - \frac{\epsilon}{2} + \frac{\epsilon^2}{8} + \cdots$$

### 6.3 迭代法

不预设渐近序列，从初始猜测出发迭代。对 $x^2 = 1 - \epsilon x$：

- 第0次：$x_0 = 1$
- 第1次：$x_1 = \sqrt{1 - \epsilon} = 1 - \frac{\epsilon}{2} - \frac{\epsilon^2}{8} + \cdots$
- 第2次：$x_2 = \sqrt{1 - \epsilon x_1}$，迭代下去...

优点：自动生成正确序列。缺点：高阶修正越来越困难，迭代可能不收敛。

### 6.4 例 2：奇异摄动方程 $\epsilon x^2 + x - 1 = 0$

**关键**：当 $\epsilon = 0$，方程退化为 $x - 1 = 0$，只有**一个**根。但 $\epsilon \neq 0$ 时有两个根。

精确解：
$$x = \frac{-1 \pm \sqrt{1 + 4\epsilon}}{2\epsilon}$$

展开两根：
- 正根（$O(1)$）：$x \sim 1 - \epsilon + 2\epsilon^2 + \cdots$
- 负根（"蒸发到无穷"）：$x \sim -\frac{1}{\epsilon} - 1 + \epsilon - 2\epsilon^2 + \cdots$

**奇异摄动**：当 $\epsilon \to 0$ 的过程中定性行为发生根本变化（方程降阶）。

### 6.5 例 3：非整数幂展开 $(1-\epsilon)x^2 - 2x + 1 = 0$

$\epsilon = 0$ 时 $x=1$ 是**二重根**。朴素的整数幂展开 $x = 1 + \epsilon x_1 + \epsilon^2 x_2 + \cdots$ 失败（$O(\epsilon)$ 方程无法满足）。

精确解：$x = \frac{1 \pm \sqrt{\epsilon}}{1 - \epsilon} = 1 \pm \epsilon^{1/2} + \epsilon \pm \epsilon^{3/2} + \cdots$

**正确的渐近序列涉及非整数幂**：一般形式 $x = 1 + \mu_1(\epsilon) x_1 + \mu_2(\epsilon) x_2 + \cdots$，其中 $\{\mu_n(\epsilon)\}$ 是渐近序列，指数和系数需同时确定。

## 7. 摄动方法—ODE (Perturbation Methods for ODE)

### 7.1 抛体问题

物体从地面以初速 $v_0$ 竖直上抛，考虑重力随高度的变化。

**控制方程**：$m\ddot{x} = -\frac{GMm}{(R+x)^2} = -mg\frac{R^2}{(R+x)^2}$，其中 $g = GM/R^2$

**初条件**：$x(0) = 0$，$\dot{x}(0) = v_0$

若忽略重力变化（$x \ll R$）：$\ddot{x} = -g$，得抛物运动：
$$x(t) = -\frac{1}{2}gt^2 + v_0 t$$

最大高度 $v_0^2/(2g)$，飞行时间 $2v_0/g$。

### 7.2 无量纲化

特征尺度：$t_c = v_0/g$，$x_c = v_0^2/g$，$\epsilon = x_c/R = v_0^2/(gR) \ll 1$

无量纲变量：$y = x/x_c$，$\tau = t/t_c$

**无量纲 ODE**：$\ddot{y}(\tau) = -\frac{1}{(1 + \epsilon y)^2}$，初条件 $y(0)=0$，$\dot{y}(0)=1$

### 7.3 正则摄动求解

展开 $y = y_0 + \epsilon y_1 + \epsilon^2 y_2 + \cdots$

展开右边：$-\frac{1}{(1+\epsilon y)^2} = -1 + 2\epsilon y - 3\epsilon^2 y^2 + \cdots$

- $O(1)$：$\ddot{y}_0 = -1$，$y_0(0)=0$，$\dot{y}_0(0)=1$
  $\Rightarrow y_0(\tau) = -\tau^2/2 + \tau$

- $O(\epsilon)$：$\ddot{y}_1 = 2y_0 = -\tau^2 + 2\tau$，$y_1(0)=0$，$\dot{y}_1(0)=0$
  $\Rightarrow y_1(\tau) = -\tau^4/12 + \tau^3/3$

一阶解：$y(\tau) = -\frac{\tau^2}{2} + \tau + \epsilon\left(-\frac{\tau^4}{12} + \frac{\tau^3}{3}\right) + O(\epsilon^2)$

### 7.4 修正飞行时间

解 $y(\tau) = 0$，假设 $\tau = 2 + \epsilon \tau_1 + \cdots$，代入得 $\tau_1 = 4/3$。

$$\tau = 2 + \frac{4}{3}\epsilon + O(\epsilon^2)$$

### 7.5 正则摄动的特点

1. 零阶解来自**无摄动问题**（$\epsilon = 0$）
2. 非线性→线性序列，高阶项由低阶决定
3. 可人为引入 $\epsilon$，最后设 $\epsilon = 1$（如 $y'' = f(x)y$，设 $\epsilon y'' = f(x)y$）
4. **一致有效性 (Uniform Validity)**：误差对全体 $x$ 同时小，只需 $\epsilon$ 够小

反例：$f(x,\epsilon) = e^{-x/\epsilon} + x$，在 $x = 10^{-2}$ 需要 $\epsilon \ll 10^{-2}$，在 $x = 10^{-6}$ 需要 $\epsilon \ll 10^{-6}$——对不同的 $x$ 需要不同程度的 $\epsilon$，不是一致有效的。

## 8. 奇异摄动方法 (Singular Perturbation) 🔥

### 8.1 线性 ODE 的边界层

**主例**：$\epsilon y'' + (1+\epsilon)y' + y = 0$，$y(0)=0$，$y(1)=1$

**精确解**（常数系数→指数拟设 $y = e^{\lambda x}$）：
$\epsilon\lambda^2 + (1+\epsilon)\lambda + 1 = 0 \Rightarrow (\lambda+1)(\epsilon\lambda+1) = 0 \Rightarrow \lambda = -1, -1/\epsilon$

$$y = c_1 e^{-x} + c_2 e^{-x/\epsilon}$$

由边界条件：
$$y(0)=0 \Rightarrow c_1 = -c_2，\quad y(1)=1 \Rightarrow c_1(e^{-1} - e^{-1/\epsilon}) = 1$$

$$y(x) = \frac{e^{-x} - e^{-x/\epsilon}}{e^{-1} - e^{-1/\epsilon}}$$

**关键观察**：$e^{-1/\epsilon}$ 是**超越小**的（$\epsilon \to 0$），没有 Taylor 展开。这是奇异摄动的标志。

对固定 $x > 0$：$y(x) \to e^{1-x}$（外解）。但收敛**不是一致的**——要 $e^{-x/\epsilon} < \delta$，需 $x > \epsilon \ln(1/\delta)$。

### 8.2 匹配渐近展开 (Matched Asymptotic Expansion)

#### 外解（通常容易，正则摄动）

设 $\epsilon = 0$：$y_0' + y_0 = 0 \Rightarrow y_{\text{out}} = c e^{-x}$

只使用 $x=1$ 的边界条件：$y_{\text{out}}(1) = 1 \Rightarrow c = e$，故 $y_{\text{out}} = e^{1-x}$

高阶：$y_n = 0$（$n \ge 1$）。

#### 内解（边界层，伸缩变换）

边界层在 $x=0$ 附近，宽度 $O(\epsilon)$。引入**伸缩变量**：

$$X = \frac{x}{\epsilon}$$

导数变换：
$$y' = \frac{dy}{dx} = \frac{dY}{dX}\frac{dX}{dx} = \frac{1}{\epsilon} Y_X, \quad y'' = \frac{1}{\epsilon^2} Y_{XX}$$

代入原 ODE：
$$\frac{1}{\epsilon} Y_{XX} + \frac{1+\epsilon}{\epsilon} Y_X + Y = 0$$

乘以 $\epsilon$：
$$Y_{XX} + (1+\epsilon)Y_X + \epsilon Y = 0$$

正则摄动展开：
- $O(1)$：$Y_{0,XX} + Y_{0,X} = 0 \Rightarrow Y_0 = C e^{-X} + D$

使用 $x=0$ 的边界条件：$Y_0(0) = 0 \Rightarrow C + D = 0 \Rightarrow Y_{\text{in}} = C(e^{-X} - 1)$

#### 匹配

$$\lim_{X \to \infty} Y_{\text{in}} = \lim_{x \to 0} y_{\text{out}}$$

$$-C = e \Rightarrow C = -e$$

$$Y_{\text{in}} = -e(e^{-X} - 1) = e - e^{1-X} = e - e^{1-x/\epsilon}$$

#### 合成解 (Composite Solution)

$$y_{\text{match}} = y_{\text{in}} + y_{\text{out}} - y_{\text{overlap}}$$

重叠区 $y_{\text{overlap}} = e$（两个解在匹配区都趋于的值）：

$$y_{\text{match}} = (e - e^{1-x/\epsilon}) + e^{1-x} - e = e^{1-x} - e^{1-x/\epsilon}$$

与精确解一致！

## 9. 边界层位置与厚度 (Boundary-Layer Location) 🔥

### 9.1 例 2：$\epsilon y'' - y' + y = 0$

BC：$y(0)=0$，$y(1)=1$

#### 尝试边界层在 $x=0$

外解（$\epsilon=0$）：$-y_0' + y_0 = 0 \Rightarrow y_0 = A e^x$

用 $x=1$ 的 BC：$A e^1 = 1 \Rightarrow y_{\text{out}} = e^{x-1}$

内解（伸缩 $X = x/\epsilon$）：
$$Y_{XX} - Y_X + \epsilon Y = 0$$

$O(1)$：$Y_{0,XX} - Y_{0,X} = 0 \Rightarrow Y_0 = B + C e^X$

$Y_0(0) = 0 \Rightarrow B + C = 0 \Rightarrow Y_{\text{in}} = B(1 - e^X)$

匹配时 $Y_{\text{in}}$ 指数增长 → **无法匹配**。边界层位置猜错了。

#### 正确位置：边界层在 $x=1$

外解：$y_0 = c e^x$，用 $x=0$ 的 BC：$y_0(0) = 0 \Rightarrow c = 0$，故 $y_{\text{out}} = 0$

伸缩 $X = (1-x)/\epsilon$（注意方向！$x=1$ 是右端点）：
$$Y_{XX} + Y_X + \epsilon Y = 0$$

$O(1)$：$Y_{0,XX} + Y_{0,X} = 0 \Rightarrow Y_0 = C + D e^{-X}$

$Y_0(0) = 1 \Rightarrow C + D = 1$

匹配：$\lim_{X \to \infty} Y_0 = C = \lim_{x \to 1} y_{\text{out}} = 0 \Rightarrow C = 0, D = 1$

$$Y_{\text{in},0} = e^{-X}$$

合成解：$y_{\text{match}} = e^{-(1-x)/\epsilon}$

### 9.2 一般规则

对 $\epsilon y'' + a(x)y' + b(x)y = 0$：

- 若 $a(x) > 0$：边界层在 **$x = 0$**（左边界）
- 若 $a(x) < 0$：边界层在 **$x = 1$**（右边界）

## 10. 内层 / 角层 (Interior / Corner Layer)

### 10.1 当 $a(x)$ 变号

**例 3**：$\epsilon y'' + xy' - y = 0$，$x \in [-1, 1]$，$y(-1)=1$，$y(1)=2$

$a(x) = x$：
- $x \in [-1, 0)$：$a(x) < 0$ → 期望边界层在右端（即 $x=0$ 方向）
- $x \in (0, 1]$：$a(x) > 0$ → 期望边界层在左端（即 $x=0$ 方向）

→ 边界层合并到 **$x=0$**，形成**内层/角层**。

### 10.2 外解

设 $\epsilon = 0$：$x y_0' - y_0 = 0 \Rightarrow \frac{dy_0}{y_0} = \frac{dx}{x} \Rightarrow y_0 = cx$

- 左端 BC：$y(-1)=1 \Rightarrow y_{\text{out}}^L = -x$
- 右端 BC：$y(1)=2 \Rightarrow y_{\text{out}}^R = 2x$

### 10.3 内解——"厚"边界层

伸缩 $X = x/\delta$。由于 $y' = \frac{1}{\delta}Y_X$，$y'' = \frac{1}{\delta^2}Y_{XX}$，$x = \delta X$，代入得：
$$\frac{\epsilon}{\delta^2} Y_{XX} + X Y_X - Y = 0$$

主导平衡要求三项同阶：$\frac{\epsilon}{\delta^2} = 1 \Rightarrow \delta = \epsilon^{1/2}$

这比通常边界层（$\delta = \epsilon$）更**厚**。

内层 ODE：$Y_{XX} + XY_X - Y = 0$（**不允许丢弃任何项**）

### 10.4 求解与匹配

$Y = X$ 是一个解。用降阶法求出第二解：

$$Y_{\text{in}} = C_1 X + C_2\left[e^{-X^2/2} + X\sqrt{\frac{\pi}{2}}\text{erf}\left(\frac{X}{\sqrt{2}}\right)\right]$$

匹配得 $C_1 = 1/2$，$C_2 = -3/(2\sqrt{2\pi})$。

## 11. 非线性系统中的边界层 (Boundary Layer in Nonlinear Systems)

### 11.1 问题设置

$\epsilon y'' = y y' - y$，$y(0)=1$，$y(1)=-1$

非线性系统——**没有现成理论判断边界层位置**。使用**几何方法（相平面分析）**。

### 11.2 相平面分析

令 $z = y'$，化为二维一阶系统：

$$\begin{cases} y' = z \\ z' = \frac{1}{\epsilon} y(z - 1) \end{cases}$$

将 $x$ 视为"时间"，$(y, z)$ 为相平面。轨迹表示解。

**关键**：
- $z = 1$ 时，$z' = 0$（无垂直速度）
- $y = 0$ 时，$y' = 0$（无水平速度）
- $y \to -y$ 对称性 → 轨迹关于 $y$ 轴对称

**快/慢区域划分**：
- $z$ 不接近 $1$：$z - 1 = O(1)$，$z' = O(1/\epsilon)$ → **快变区**（边界层）
- $z \approx 1$：$z - 1 = O(\epsilon)$，$z' = O(1)$ → **慢变区**（外解）

几何图形表明边界层在 **$x = 1/2$**。

### 11.3 匹配渐近展开

**外解**（慢变区，$z \approx 1$）：$\epsilon = 0 \Rightarrow y_0(y_0' - 1) = 0$
- $y_0' = 1$（非平凡）$\Rightarrow y_0 = x + C$
- 左 BC：$y(0)=1 \Rightarrow y_{\text{out},L} = x + 1$
- 右 BC：$y(1)=-1 \Rightarrow y_{\text{out},R} = x - 2$

**内解**（边界层，$x = 1/2$ 附近）：伸缩 $X = (x-1/2)/\epsilon$

代入得 $\ddot{Y} = Y\dot{Y} - Y$（去掉了 $\epsilon$）

正则摄动 $O(1)$：$\ddot{Y}_0 = Y_0 \dot{Y}_0$

注意到 $\ddot{Y}_0 = \frac{d}{dX}(\frac{1}{2}Y_0^2) = Y_0 \dot{Y}_0$。由对称性 $Y_0(0) = 0$。

对 $\ddot{Y}_0 = Y_0 \dot{Y}_0$ 积分一次：
$$\dot{Y}_0 = \frac{1}{2}Y_0^2 + B, \quad B < 0$$

令 $B = -b^2/2$，分离变量：
$$\frac{\dot{Y}_0}{Y_0^2 - b^2} = \frac{1}{2} \Rightarrow Y_0(X) = -b \tanh\left(\frac{bX}{2}\right)$$

**匹配**：
$$\lim_{x \to 1/2^-} y_{\text{out},L} = \frac{3}{2} = b \Rightarrow b = \frac{3}{2}$$

**合成解**：
$$y_{\text{match}} = x + 1 - \frac{3}{2}\tanh\left(\frac{3}{4}\frac{x-1/2}{\epsilon}\right) - \frac{3}{2} + \cdots$$

# 第三部分②：WKB 方法 — 全局坍塌

> **失效类型**：$\epsilon^2 y'' = Q(x)y$ 在 $Q(x)$ 变号处全局失效
> **与边界层的区别**：边界层处理**局部**快速变化，WKB 处理**全局**缓变失效（调制振荡/指数衰减）
> **统一性**：$y = e^{S/\delta}$ 拟设 = 主导平衡的 $\epsilon$ 参数化版本
>
> ### 🔥 考试重点（老师强调）
> **WKB 全套流程**：Ansatz $y = e^{S/\epsilon}$ → $S_0' = \pm\sqrt{Q}$ → $S_1' = -Q'/4Q$ → 通解 → **Turning Point** $Q(x_0)=0$ 处 WKB 失效 → 内层伸缩 $\epsilon^{2/3}$ → **Airy 方程** → Airy 函数渐近连接两侧 → **双转折点 Bohr-Sommerfeld 量子化** $\oint\sqrt{Q}dx = (n+1/2)\pi\epsilon$
>
> **老师口述的 WKB 逻辑链（重点）**：
> 1. 会推 WKB（从 $y = e^{S/\epsilon}$ 出发，主导平衡逐阶求解）
> 2. 知道 **turning point** 是什么（$Q(x_0)=0$），为什么 WKB 不 work（$1/Q^{1/4}$ 发散）
> 3. **WKB 不 work 的时候 → 重新回到 boundary layer**：turning point 附近有一个 **inner layer**，要专门对它处理
> 4. 处理过程中要知道**怎么得到 thickness**（主导平衡 → $\delta = \epsilon^{2/3}$）
> 5. Airy 函数连接公式：$x>x_0$ 指数衰减（有界性 → 只留 Ai），$x<x_0$ 振荡

## 12. WKB 方法 🔥

### 12.1 定位

- **边界层**：处理**局部**快速变化（匹配内外解）
- **WKB**：处理**全局**缓变失效（调制振荡/指数衰减）

### 12.2 基本方程

$\epsilon^2 y'' = Q(x) y$（$\epsilon \to 0$），等价于 Schrödinger 方程：
$$i\hbar\frac{\partial\psi}{\partial t} = -\frac{\hbar^2}{2m}\frac{\partial^2\psi}{\partial x^2} + V(x)\psi$$

### 12.3 主导平衡推导

假设 $y = e^{S/\epsilon}$，$S = S_0 + \epsilon S_1 + \epsilon^2 S_2 + \cdots$

求导：
$$y' = \frac{S'}{\epsilon} e^{S/\epsilon}$$
$$y'' = \left[\frac{S''}{\epsilon} + \frac{(S')^2}{\epsilon^2}\right] e^{S/\epsilon} = \frac{1}{\epsilon^2}\left[\epsilon S'' + (S')^2\right] e^{S/\epsilon}$$

代入 ODE：$\epsilon S'' + (S')^2 - Q(x) = 0$

代入 $S$ 的展开并按 $\epsilon$ 幂次收集：

- **$O(1)$**：$(S_0')^2 = Q$
  $$\Rightarrow S_0' = \pm \sqrt{Q}, \quad S_0 = \pm \int_0^x \sqrt{Q(s)} ds$$

- **$O(\epsilon)$**：$S_0'' + 2S_0' S_1' = 0$
  $$\Rightarrow S_1' = -\frac{S_0''}{2S_0'} = -\frac{Q'}{4Q} \Rightarrow S_1 = -\frac{1}{4}\ln Q + \text{const}$$

- 更高阶：$S_1'' + (S_1')^2 + 2S_0' S_2' = 0$ → 可逐阶求解

### 12.4 WKB 通解

$$y = \frac{e^{S/\epsilon}}{Q^{1/4}}\left[C_1 e^{\frac{1}{\epsilon}\int\sqrt{Q}dx} + C_2 e^{-\frac{1}{\epsilon}\int\sqrt{Q}dx}\right]$$

根据 $Q(x)$ 的符号：

| $Q(x) > 0$（振荡型） | $Q(x) < 0$（指数型） |
|---|---|
| $y \sim \frac{1}{Q^{1/4}}\left[c_1\cos\left(\frac{1}{\epsilon}\int\sqrt{Q}dx\right) + c_2\sin\left(\frac{1}{\epsilon}\int\sqrt{Q}dx\right)\right]$ | $y \sim \frac{1}{(-Q)^{1/4}}\left[c_3 e^{-\frac{1}{\epsilon}\int\sqrt{-Q}dx} + c_4 e^{\frac{1}{\epsilon}\int\sqrt{-Q}dx}\right]$ |

### 12.5 例：$x'' + e^{-t} x = 0$

设 $\epsilon^2 \ddot{x} + e^{-t}x = 0$，$\omega = e^{-t/2}$ 缓慢衰减。$t \gg 1/\epsilon$ 时 $\omega \to 0$，振荡停止。

换元 $T = \epsilon t$，$X(T) = x(t)$，$\ddot{x} = \epsilon^2 \ddot{X}$：

$\epsilon^2 \ddot{X} + e^{-T} X = 0$。用 WKB：
$$X \sim e^{T/4}\left[c_1\sin\left(\frac{2}{\epsilon}e^{-T/2}\right) + c_2\cos\left(\frac{2}{\epsilon}e^{-T/2}\right)\right]$$

注意：WKB 也可以重新推导出边界层的结果（如 8.1 节的例）。

### 12.6 转折点 (Turning Point)

**Q(x₀) = 0**，解从振荡型变为指数型。$1/Q^{1/4}$ 发散 → WKB 在 $x_0$ 附近失效。

**WKB 有效的条件**：$(S_0')^2 / \epsilon \gg S_0''$，即 $|Q| \gg \sqrt{\epsilon} |Q'|$。

对于 $Q \approx a(x-x_0)$，失效区域：$|x - x_0| < O(\epsilon^{2/3})$。

### 12.7 Airy 连接公式

在转折点附近引入内层伸缩：$x - x_0 = \epsilon^{2/3} Z$

得 Airy 方程：$Y_{ZZ} = ZY$

**Airy 函数的渐近行为**：

| 区域 | $\text{Ai}(Z)$ | $\text{Bi}(Z)$ |
|------|----------------|----------------|
| $Z \to +\infty$ | $\frac{1}{2\sqrt{\pi}}Z^{-1/4}e^{-\frac{2}{3}Z^{3/2}}$（衰减） | $\frac{1}{\sqrt{\pi}}Z^{-1/4}e^{\frac{2}{3}Z^{3/2}}$（增长） |
| $Z \to -\infty$ | $\frac{1}{\sqrt{\pi}}(-Z)^{-1/4}\sin\left(\frac{2}{3}(-Z)^{3/2} + \frac{\pi}{4}\right)$ | $\frac{1}{\sqrt{\pi}}(-Z)^{-1/4}\cos\left(\frac{2}{3}(-Z)^{3/2} + \frac{\pi}{4}\right)$ |

**若物理上要求解在 $x > x_0$ 侧有界（如 Schrödinger 方程的束缚态）→ 必须丢弃 Bi（因为 Bi 在 $Z \to +\infty$ 时指数增长）→ 只保留 Ai**。

**单转折点匹配**（设 $x \to -\infty$ 无界 → 必须用 Ai）：

- $x < x_0$（$Q < 0$，振荡）:
  $$y_L \sim \frac{2C_3}{(-Q)^{1/4}}\sin\left(\frac{1}{\epsilon}\int_x^{x_0}\sqrt{-Q}dt + \frac{\pi}{4}\right)$$

- $x > x_0$（$Q > 0$，指数衰减）:
  $$y_R \sim \frac{C_3}{Q^{1/4}}e^{-\frac{1}{\epsilon}\int_{x_0}^x\sqrt{Q}dt}$$

$C_3$ 通过 Airy 函数匹配确定。

## 13. 特征值问题 (Eigenvalue Problem) 🔥

### 13.1 基本概念

ODE 仅在参数取特定值（**特征值**）时才有非平凡解（**特征函数**）。

**例 1**：$y'' + E y = 0$，$y(0) = y(\pi) = 0$

通解：$y = \sin(\sqrt{E} x)$（已满足 $y(0)=0$）

$y(\pi) = 0 \Rightarrow \sin(\sqrt{E}\pi) = 0 \Rightarrow \sqrt{E}\pi = n\pi$

$$E_n = n^2, \quad y_n = \sin(nx), \quad n = 1, 2, 3, \ldots$$

### 13.2 例 2：超越特征值

$y'' + (E - x)y = 0$，$y(0) = y(\pi) = 0$

改写：$y'' = (x - E)y$。令 $X = x - E$，得 $Y'' = XY$（Airy 方程）。

通解：$y = \text{Ai}(x - E)$

$y(0) = 0 \Rightarrow \text{Ai}(-E) = 0$。$E$ 是 $\text{Ai}(-z)$ 的零点：$E_1 \approx 2.3$，$E_2 \approx 4$，$E_3 \approx 5.5$，...

### 13.3 WKB 近似大特征值

$\epsilon^2 y'' + Q(x) y = 0$（$\epsilon = 1/\sqrt{E} \ll 1$），$Q > 0$，$y(0) = y(\pi) = 0$

WKB 振荡解：$y \sim \frac{1}{Q^{1/4}}\left[c_1\cos\left(\frac{1}{\epsilon}\int\sqrt{Q}dx\right) + c_2\sin\left(\frac{1}{\epsilon}\int\sqrt{Q}dx\right)\right]$

$y(0)=0 \Rightarrow c_1 = 0$。$y(\pi)=0 \Rightarrow \sin\left(\frac{1}{\epsilon}\int_0^\pi\sqrt{Q}dx\right) = 0$

$$\frac{1}{\epsilon}\int_0^\pi\sqrt{Q}dx = n\pi \Rightarrow E_n \sim \frac{n^2\pi^2}{\left(\int_0^\pi\sqrt{Q}dx\right)^2}$$

### 13.4 Sturm-Liouville 理论

一般二阶特征值问题可化为 S-L 形式：

$$\frac{d}{dx}\left(p(x)\frac{dy}{dx}\right) + q(x)y = -\lambda w(x)y$$

其中 $p > 0$，$w > 0$（在考虑的区间内）。在适当边界条件下：
- 有无穷多特征值 $\lambda_0 < \lambda_1 < \lambda_2 < \cdots$，均为实数
- 特征函数 $\{y_n\}$ 关于权函数 $w(x)$ **正交归一**：$\int y_n(x) y_m(x) w(x) dx = \delta_{nm}$
- 若 $\{y_n\}$ 完备，则任意函数可展开：$y(x) = \sum c_n y_n(x)$，$c_n = \int y(x) y_n(x) w(x) dx$

## 14. Schrödinger 方程与双转折点

### 14.1 问题

$\epsilon^2 y'' = (V(x) - E) y$，$y(\pm\infty) \to 0$

$Q(x) = V(x) - E$。对于势阱 $V(x)$，能量 $E$ 与 $V(x)$ 相交于两个转折点 $A, B$（$V(A) = V(B) = E$）：

- 阱内（$A < x < B$）：$V(x) < E$ → $Q < 0$ → 振荡
- 阱外（$x < A$ 和 $x > B$）：$V(x) > E$ → $Q > 0$ → 指数衰减

### 14.2 匹配策略

1. 从 $x=B$ 用 Airy 连接：向右 $x>B$ 衰减（$\text{Ai}$），向左 $x<B$ 振荡
2. 从 $x=A$ 用 Airy 连接：向左 $x<A$ 衰减（$\text{Ai}$），向右 $x>A$ 振荡
3. 在阱内中间区域，两个振荡解需要一致匹配

### 14.3 量子化条件 (Bohr-Sommerfeld)

相位匹配要求：

$$\frac{1}{\epsilon}\int_A^B \sqrt{E - V(x)} dx = \left(n + \frac{1}{2}\right)\pi, \quad n = 0, 1, 2, \ldots$$

**直觉**：阱内的半波长数必须是 $n + 1/2$（两端的 $\pi/4$ 相移各贡献一个"半长"）。

# 第四部分：动力系统与分岔 — 定性分析的范式转换

> 从"解的具体形式"转向"解的定性行为"。前置工具：Normal Form（规范型）
> 三类经典范式：鞍结（不动点湮灭）| 跨临界（交换稳定性）| 叉形（对称破缺）
> 脉络图中此部分作为最终收敛点——所有渐近/摄动/多尺度方法汇聚到对系统定性行为的理解
>
> ### 🔥 考试重点（老师强调）
> **考试路径**（老师口述）：**在 rescaled 的方程上** → identify **bifurcation parameters** → 根据 bifurcation parameters 绘出 **bifurcation diagram** → identify 是哪一种 bifurcation（pitchfork / saddle-node / transcritical / imperfect pitchfork）。
>
> **1-D Bifurcation 标准四步**：① 选取 Bifurcation Parameter $r$ → ② 寻找 Fixed Point $f(x^*, r)=0$ → ③ 判定 Stability（$f'(x^*)$ 符号）→ ④ 绘制 Bifurcation Diagram（$x^*$ vs $r$）
>
> **三类范式必须能默写并识别**：
> - 鞍结 (Saddle-Node)：$\dot{x}=r+x^2$（两个不动点碰撞消失）
> - 跨临界 (Transcritical)：$\dot{x}=rx-x^2$（不动点交换稳定性）
> - 叉形 (Pitchfork)：$\dot{x}=rx-x^3$ 超临界 / $\dot{x}=rx+x^3$ 亚临界（对称破缺）
> - **Imperfect Pitchfork** 也要了解：$\dot{x}=s+rx-x^3$，参数空间 $(r,s)$ 中有 cusp 结构
>
> **Rescaling 与分岔的关系**：Rescaling 后得到的简化方程 → 提取 Normal Form → 识别分岔类型。所以 Rescaling（§5）是分岔分析的直接前置。
>
> - **Delayed Bifurcation**（§17）：$\Delta s \sim 2.338\,\epsilon^{2/3}$，Riccati → Airy 变换（同 §12.7 技术）

## 15. 动力系统与分岔 (Dynamical Systems and Bifurcation) 🔥

### 15.1 一维分岔

#### 鞍结分岔 (Saddle-Node)

**标准形**：$\dot{x} = x^2 + r$

| $r$ | 不动点 |
|-----|--------|
| $r < 0$ | $x = \pm\sqrt{-r}$（一稳定、一不稳定） |
| $r = 0$ | $x = 0$（**半稳**） |
| $r > 0$ | 无不动点 |

$r$ 增加穿过 0 时，两个不动点湮灭（annihilation）。

**一般情况**：$\dot{x} = f(x; r)$，在 $(x_*, r_c)$ 附近：
$$f(x; r) \approx a(r - r_c) + b(x - x_*)^2$$

#### 跨临界分岔 (Transcritical)

**标准形**：$\dot{x} = rx - x^2$

不动点：$x=0$ 和 $x=r$（恒存在）。$r$ 穿过 0 时交换稳定性。

**例**：$\dot{x} = r\ln x + x - 1$。$x=1$ 是恒不动点。令 $x = u+1$，展开：
$$\dot{u} = r\ln(u+1) + u \approx r\left(u - \frac{u^2}{2}\right) + u = (r+1)u - \frac{r}{2}u^2$$

换元到标准形：$w' = (r+1)w - w^2$

#### 叉形分岔 (Pitchfork)

**超临界标准形**：$\dot{x} = rx - x^3$

| $r$ | 不动点 |
|-----|--------|
| $r \le 0$ | $x = 0$（稳定） |
| $r > 0$ | $x = 0$（不稳定），$x = \pm\sqrt{r}$（稳定） |

**$x \to -x$ 对称性**。分岔图呈 pitchfork 形状。

**亚临界**：$\dot{x} = rx + x^3$（$\pm\sqrt{-r}$ 在 $r<0$ 时出现但**不稳定**）。

#### 不完备叉形分岔 (Imperfect Pitchfork)

$\dot{x} = s + rx - x^3$（两个参数：$r$ 与不对称 $s$）

参数空间 $(r, s)$ 中有**尖点 (cusp)** 结构。$s \neq 0$ 时，完美叉形被破坏，分岔变为鞍结型。

### 15.2 二维分岔

二维系统中不动点有更丰富的类型：

| 类型 | 特征 |
|------|------|
| **稳定结点 (Stable Node)** | 所有轨迹直接趋近 |
| **不稳定结点 (Unstable Node)** | 所有轨迹直接远离 |
| **鞍点 (Saddle)** | 一个方向趋近、一个远离 |
| **中心 (Center)** | 闭合椭圆轨道族（中性稳定） |
| **稳定/不稳定焦点 (Spiral)** | 螺旋趋近/远离 |

与一维分岔组合的标准形（$y' = -y$ 提供稳定方向）：

- 鞍结：$\dot{x} = r - x^2$，$\dot{y} = -y$
- 跨临界：$\dot{x} = rx - x^2$，$\dot{y} = -y$
- 叉形：$\dot{x} = rx - x^3$，$\dot{y} = -y$

## 16. 二维系统与极限环 (2D Systems and Limit Cycles)

### 16.1 二维系统中的新现象

除不动点外，二维系统还出现：

- **极限环**：孤立的闭合轨道（不同于中心周围的连续闭轨族）
- 极限环的稳定性：稳定（内外轨迹都趋近）、不稳定（内外都远离）、半稳

### 16.2 Van der Pol 方程与极限环

$$\ddot{x} + \epsilon(x^2 - 1)\dot{x} + x = 0$$

- $|x| < 1$：负阻尼 → 小振荡**增长**（激励）
- $|x| > 1$：正阻尼 → 大振荡**衰减**（耗散）
- 平衡 → 存在**稳定极限环**

零阶（$\epsilon = 0$）：简谐振子，$\omega = 1$。

### 16.3 Liénard 定理

对 $\ddot{x} + f(x)\dot{x} + g(x) = 0$：

- $g$ 是奇函数，$f$ 是偶函数
- $F(x) = \int_0^x f(u)du$ 满足适当条件（单零点等）
→ 存在**唯一的稳定极限环**。

## 17. 不动点的动力学 (Dynamics of Fixed Points) ⭐

### 17.1 滞后与突变 (Hysteresis / Catastrophe)

考虑慢变参数的 ODE：

$$\frac{dy}{dt} = y - \frac{y^3}{3} - s(t)$$

其中 $s(t)$ 缓慢扫描（$s = \epsilon t$）。

$s$ 为常数时，不动点满足 $s = y - y^3/3$（S 形折叠曲线）。当 $s$ 穿过临界值 $s = 2/3$，系统从稳定上分支突然跳到稳定下分支——**突变**。

### 17.2 延迟分岔 (Delayed Bifurcation)

有限扫描速率下，跳跃不是精确发生在 $s=2/3$，而是**延迟**。延迟量 $\propto \epsilon^{2/3}$。

设 $t_c = (s - 2/3)/\epsilon^{2/3}$，展开：
$$Y = 1 + \epsilon^{1/3} Y_{c1} + \epsilon^{2/3} Y_{c2} + \cdots$$

内层 ODE 演化为 **Riccati 方程**：
$$\frac{dY_{c1}}{dt_c} = -Y_{c1}^2 - t_c$$

令 $Y_{c1} = W'/W$，化为 Airy 方程 $W'' = -t_c W$。解为：
$$Y_{c1}(t_c) = -\frac{\text{Ai}'(-t_c)}{\text{Ai}(-t_c)}$$

$Y_{c1}$ 在 $\text{Ai}(-t_c) = 0$ 处发散（跳跃）。第一零点 $t_c \approx 2.338$。

$$\text{延迟 } \Delta s = 2.338 \,\epsilon^{2/3}$$

# 第三部分③：多尺度分析 — 长时间演化失效

> **失效类型**：超长区间上正则摄动展开 → 时间积累 → 相位/振幅漂移 → Secular Term（久期项）
> **核心目标**：消除久期项 → **Amplitude Equation（振幅方程）**
> **两套方法**：Poincaré-Lindstedt（仅周期解，展开频率）| 多尺度分析（周期+非周期，多重时间变量）
>
> ### 🔥 考试重点（老师强调）
> **Multiple Scale Analysis**（老师口述）：**最简单的 two timing — $t$ 和 $\epsilon t$**。掌握这个就基本掌握了整个方法。后面那些更精细的东西是锦上添花。
>
> **Two-Timing 标准步骤**：
> 1. 引入 $T_0 = t$（快尺度），$T_1 = \epsilon t$（慢尺度）
> 2. 偏导展开：$\frac{d}{dt} = \frac{\partial}{\partial T_0} + \epsilon\frac{\partial}{\partial T_1}$，$\frac{d^2}{dt^2} = \frac{\partial^2}{\partial T_0^2} + 2\epsilon\frac{\partial^2}{\partial T_0\partial T_1}$
> 3. $O(1)$ 给出 $Y_0 = A(T_1)e^{iT_0} + c.c.$
> 4. $O(\epsilon)$ 消除共振项（$e^{\pm iT_0}$ 系数为零）→ **Amplitude Equation** $A' = f(A)$
> 5. 解振幅方程 $A(T_1) = R(T_1)e^{i\theta(T_1)}$，得到慢变演化
>
> **两种方法的考试适用范围**：
> - **Poincaré-Lindstedt**：展开频率 $\omega = \omega_0 + \epsilon\omega_1 + \cdots$，仅适用于周期解
> - **多尺度 Two-Timing**：更通用，周期+阻尼+极限环均可处理
>
> **经典考题**：
> - **Duffing 方程**两类方法的统一结果：$\omega = 1 + 3\epsilon/8$
> - **Van der Pol** 的振幅方程 $R' = -R(R^2-1)/2$ → 本身是叉形分岔结构（与 §15 合流）
> - **弱阻尼振子**：$A' + A = 0 \Rightarrow A = A_0 e^{-T_1}$ → 振幅指数衰减

## 18. 受迫谐振子 (Forced Harmonic Oscillator)

### 18.1 $y'' + \omega_0^2 y = \cos(\omega_f t)$

齐次解：$y_h = A\cos(\omega_0 t) + B\sin(\omega_0 t)$

特解：
- $\omega_0 \neq \omega_f$：$y_p = \frac{\cos(\omega_f t)}{\omega_0^2 - \omega_f^2}$
- $\omega_0 = \omega_f$（共振）：**长期项**
  $$y_p = \frac{t}{2\omega_f} \sin(\omega_f t)$$
  振幅随时间线性增长。

**长期项的标志**：强迫项本身是齐次方程的解。

<a id="sec-19"></a>
## 19. 非线性弹簧与 Duffing 振子 ⭐

### 19.1 Duffing 方程

$$\ddot{y} + y + \epsilon y^3 = 0, \qquad y(0)=1,\; \dot{y}(0)=0$$

- $\epsilon > 0$：硬化弹簧
- $\epsilon < 0$：软化弹簧
- 无阻尼、无驱动 → 预期周期解

### 19.2 正则摄动的失败

朴素展开 $y = y_0 + \epsilon y_1 + \epsilon^2 y_2 + \cdots$

- $O(1)$：$\ddot{y}_0 + y_0 = 0$，$y_0(0)=1$，$\dot{y}_0(0)=0 \Rightarrow y_0 = \cos t$

- $O(\epsilon)$：$\ddot{y}_1 + y_1 = -y_0^3 = -\cos^3 t$

用三角恒等式：$\cos^3 t = \frac{1}{4}\cos 3t + \frac{3}{4}\cos t$

$\ddot{y}_1 + y_1 = -\frac{1}{4}\cos 3t - \frac{3}{4}\cos t$

$\cos t$ 项是**共振项**→产生**长期项**：$y_1 \sim -\frac{3}{8}t\sin t$

**问题**：
- 物理解应是周期的，长期项破坏了周期性
- 当 $t = O(1/\epsilon)$，$\epsilon y_1$ 与 $y_0$ 同阶 → 渐近性失效
- 所有高阶都会出现长期项

**根源**：Duffing 项改变了振荡频率（不再是 $\omega=1$），但正则摄动强迫零阶解以 $\omega=1$ 振荡，导致共振。

<a id="sec-20"></a>
## 20. Poincaré-Lindstedt 方法 ⭐

### 20.1 基本思想

**不仅展开 $y$，也展开频率 $\omega$**。引入伸缩时间 $\tau = \omega t$：

$$\omega = \omega_0 + \epsilon\omega_1 + \epsilon^2\omega_2 + \cdots$$

ODE 变为：$\omega^2 Y'' + Y + \epsilon Y^3 = 0$（$Y' = dY/d\tau$）

### 20.2 Duffing 方程的 P-L 求解

展开 $Y = Y_0 + \epsilon Y_1 + \epsilon^2 Y_2 + \cdots$

- $O(1)$：$\omega_0^2 Y_0'' + Y_0 = 0$，$Y_0(0)=1$，$Y_0'(0)=0$
  物理上 $\epsilon=0$ 时即简谐振子 → $\omega_0 = 1$，$Y_0 = \cos\tau$

- $O(\epsilon)$：$Y_1'' + Y_1 = -2\omega_1 Y_0'' - Y_0^3$
  代入 $Y_0 = \cos\tau$（$Y_0'' = -\cos\tau$）及 $\cos^3\tau = \frac{1}{4}\cos 3\tau + \frac{3}{4}\cos\tau$：
  $$Y_1'' + Y_1 = 2\omega_1\cos\tau - \frac{1}{4}\cos 3\tau - \frac{3}{4}\cos\tau$$
  $$= -\frac{1}{4}\cos 3\tau + \left(2\omega_1 - \frac{3}{4}\right)\cos\tau$$

**消除共振条件**：令 $\cos\tau$ 系数为零：
$$2\omega_1 - \frac{3}{4} = 0 \Rightarrow \omega_1 = \frac{3}{8}$$

剩下 $Y_1'' + Y_1 = -\frac{1}{4}\cos 3\tau$，特解 $Y_{1p} = \frac{1}{32}\cos 3\tau$

由初条件得 $Y_1 = \frac{1}{32}(\cos 3\tau - \cos\tau)$

### 20.3 最终结果

$$y(t) \approx \cos\left(\left(1 + \frac{3\epsilon}{8}\right)t\right) + \frac{\epsilon}{32}\left[\cos\left(3\left(1 + \frac{3\epsilon}{8}\right)t\right) - \cos\left(\left(1 + \frac{3\epsilon}{8}\right)t\right)\right]$$

时间伸缩 $\tau = \omega t = (1 + 3\epsilon/8)t$ → 频率得到了 $\epsilon$ 修正！

## 21. 多尺度分析 (Multiple Scale Analysis) 🔥

### 21.1 基本思想

引入多个时间尺度，**视为独立变量**：

$$T_0 = t \;\text{(快)}, \quad T_1 = \epsilon t \;\text{(慢)}, \quad T_2 = \epsilon^2 t \;\text{(更慢)}, \ldots$$

**物理直觉**：解是一个快速振荡（$T_0$），其振幅/相位在慢时间 $T_1 = \epsilon t$ 上缓慢调制。

导数展开：
$$\frac{d}{dt} = \frac{\partial}{\partial T_0} + \epsilon\frac{\partial}{\partial T_1} + \cdots$$

$$\frac{d^2}{dt^2} = \frac{\partial^2}{\partial T_0^2} + 2\epsilon\frac{\partial^2}{\partial T_0\partial T_1} + \epsilon^2\frac{\partial^2}{\partial T_1^2} + \cdots$$

### 21.2 Duffing 方程的多尺度分析

展开 $Y = Y_0 + \epsilon Y_1 + \cdots$

- $O(1)$：$\frac{\partial^2 Y_0}{\partial T_0^2} + Y_0 = 0$（关于 $T_0$ 的简谐振子）
  $$\Rightarrow Y_0 = A(T_1) e^{iT_0} + \bar{A}(T_1) e^{-iT_0}$$

- $O(\epsilon)$：$\frac{\partial^2 Y_1}{\partial T_0^2} + Y_1 = -Y_0^3 - 2\frac{\partial^2 Y_0}{\partial T_0\partial T_1}$

  右边产生共振项（$\propto e^{\pm iT_0}$）。为使解有界（消除长期项），共振项系数必须为零：

  $$3A^2\bar{A} + 2i A' = 0 \quad \text{其中 } A' = \frac{dA}{dT_1}$$

**振幅方程**：令 $A(T_1) = R(T_1) e^{i\theta(T_1)}$，代入 $A' = (R' + iR\theta')e^{i\theta}$，得 $3R^3 + 2iR' - 2R\theta' = 0$。分离实虚部：

- **虚部**：$2R' = 0 \Rightarrow R' = 0 \Rightarrow R = R_0$（振幅不变——无阻尼）
- **实部**：$-2R\theta' + 3R^3 = 0 \Rightarrow \theta' = \frac{3}{2}R_0^2$

故 $A(T_1) = R_0 e^{i(3R_0^2 T_1/2 + \theta_0)}$

代回：
$$Y_0 = 2R_0\cos\left(T_0 + \frac{3}{2}R_0^2 T_1 + \theta_0\right)$$

初条件 $y(0)=1$，$y'(0)=0 \Rightarrow R_0 = 1/2$，$\theta_0 = 0$：

$$y \approx \cos\left(t + \frac{3\epsilon}{8}t\right) = \cos\left(\left(1 + \frac{3\epsilon}{8}\right)t\right)$$

**与 P-L 方法一致！**

### 21.3 弱阻尼振子

$\ddot{y} + 2\epsilon\dot{y} + y = 0$，$y(0)=1$，$\dot{y}(0)=0$

精确解：$y = A e^{-\epsilon t} \cos(\sqrt{1-\epsilon^2} t + \phi)$

**正则摄动**：$y_0 = \cos t$，$y_1$ 出现长期项 $y_1 = -t\cos t$

**多尺度分析**：
- $O(1)$：$Y_0 = A(T_1)e^{iT_0} + c.c.$
- $O(\epsilon)$：消除共振 → $A' + A = 0 \Rightarrow A(T_1) = A_0 e^{-T_1}$
- $Y_0 = e^{-\epsilon t}\cos t$（到 $O(\epsilon)$ 与精确解一致）

### 21.4 多尺度 vs P-L 方法

| P-L | 多尺度 |
|-----|--------|
| 一个伸缩时间 $\tau = \omega t$ | 多个独立时间变量 $T_0, T_1, T_2, \ldots$ |
| 调整 $\omega$ 消除长期项 | 消除共振保证解有界 |
| 适用于周期运动 | 也适用于渐近趋近（阻尼、极限环） |
| 实际上等价于二尺度分析 | 更一般、更灵活 |

<a id="sec-22"></a>
## 22. Van der Pol 方程 ⭐

### 22.1 方程与物理

$$\ddot{y} + \epsilon(y^2 - 1)\dot{y} + y = 0$$

- **非线性阻尼** $\epsilon(y^2 - 1)\dot{y}$：$|y| < 1$ 时负阻尼（激励），$|y| > 1$ 时正阻尼（耗散）
- $\epsilon = 0$ → 简谐振子
- $\epsilon > 0$ → 存在**稳定极限环**，振幅 $\approx 2$

### 22.2 多尺度分析

快尺度 $T_0 = t$，慢尺度 $T_1 = \epsilon t$

- $O(1)$：$Y_0 = A(T_1) e^{iT_0} + c.c.$

- $O(\epsilon)$：消除 $e^{iT_0}$ 共振项 → **振幅方程**：
  $$2A' = -A(|A|^2 - 1)$$

令 $A = R e^{i\theta}$：

虚部：$\theta' = 0 \Rightarrow \theta = \theta_0$（常数相位）

实部：$2R' = -R(R^2 - 1)$

### 22.3 振幅的动力学

$$\frac{dR}{dT_1} = -\frac{R}{2}(R^2 - 1)$$

- $R = 0$：**不稳定**不动点
- $R = 1$：**稳定**不动点（→ 振幅 $2R = 2$）
- 无论初始振幅如何，$R \to 1$（$T_1 \to \infty$）

**解析积分**：令 $u = 1/R^2$，得 $\dot{u} = u - 1$，解：

$$R(T_1) = \frac{1}{\sqrt{1 + \left(\frac{1}{R_0^2} - 1\right)e^{-T_1}}}$$

**零阶解**：

$$y(t) \approx 2R(\epsilon t) \cos(t + \theta_0) = \frac{2\cos(t + \theta_0)}{\sqrt{1 + \left(\frac{1}{R_0^2} - 1\right)e^{-\epsilon t}}}$$

当 $t \to \infty$，分母 $\to 1$，$y \to 2\cos(t + \theta_0)$——极限环振幅为 2。

### 22.4 极限环形成的物理机制

半个周期在 $|y| < 1$（激励），半个周期在 $|y| > 1$（耗散），恰好平衡 → 稳定极限环。

## 补充 1：Rayleigh-Lorentz 振子 ⭐

> 老师要求：了解核心概念 —— 能量不守恒，绝热不变量 (adiabatic invariant)

### 方程

$$\ddot{x} + \epsilon(\dot{x}^2 - 1)\dot{x} + x = 0$$

与 Van der Pol 的区别：阻尼项为 $\epsilon(\dot{x}^2 - 1)\dot{x}$（依赖速度平方），而非 $\epsilon(x^2 - 1)\dot{x}$（依赖位移平方）。

### 核心物理概念

- **能量不守恒**：系统是非保守的，阻尼非线性导致能量在周期内变化
- **绝热不变量 (Adiabatic Invariant)**：当参数缓慢变化时，作用量 $J = \oint p\,dq$ 近似守恒。在振动系统中，$E/\omega \approx$ 常数
- 与 Van der Pol 的联系：两者都产生**稳定极限环**，但阻尼机制不同

### 考试要点
- 理解能量不守恒的物理含义
- 理解 adiabatic invariant 的基本概念：慢变参数下 $E/\omega$ 保持（近似）不变
- 不要求完整求解，但要知道和 Van der Pol/Duffing 的关系

## 补充 2：离散化 (Discretization) ⭐

> 老师要求：掌握基础的将连续问题离散化的能力

### 核心思想

将连续 ODE/PDE 转化为离散的差分方程，以便数值求解或分析。

### 基本方法

**一阶导数**：
- 向前差分：$y'(x) \approx \frac{y(x+h) - y(x)}{h}$
- 向后差分：$y'(x) \approx \frac{y(x) - y(x-h)}{h}$
- 中心差分：$y'(x) \approx \frac{y(x+h) - y(x-h)}{2h}$（$O(h^2)$）

**二阶导数**：
$$y''(x) \approx \frac{y(x+h) - 2y(x) + y(x-h)}{h^2}$$

### 在课程中的应用

1. **边值问题**：$y'' = f(x,y,y')$ → 网格点 $x_i$ → 差分方程 → 三对角线性系统
2. **特征值问题**：$y'' = \lambda y$ → 离散化 → $Ay = \lambda y$（矩阵特征值）
3. **分岔分析**：连续 ODE $\dot{x} = f(x;r)$ → 不动点迭代 $x_{n+1} = F(x_n;r)$

### 考试要点
- 能够将简单的二阶 ODE 边值问题写成差分格式
- 理解 $h$ 的选取对精度的影响（$O(h)$ vs $O(h^2)$）
- 了解离散化在分岔分析中作为迭代工具的应用
