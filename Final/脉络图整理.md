## 一、渐近理论根基（Asymptotic Fundamentals）

> y ≈ 85–200 | 全局逻辑出发点的三层地基

### 1. 渐近序列（Asymptotic Sequence）
- **核心问题**：渐近构造的合法性基石
- 定义：$\{\phi_n\}$ 满足 $\phi_{n+1} = o(\phi_n)$
- 标注：$O(1), O(\epsilon), O(\epsilon^2), \ldots$

### 2. 渐近展开（Asymptotic Expansion）
- **核心问题**：展开的 R 余项控制
- **特征**：无法用幂级数 $\{1, x, x^2, \ldots\}$ 表示 → "摄动失效" → 引出奇异摄动方法

### 3. 超越小量（Transcendental Smallness）★ 全局逻辑源头
- **特征**：$e^{-1/\epsilon}$ 型 — 比任何 $\epsilon^n$ 都小，但不可忽略
- **原理**：指数放大效应 → 物理上贡献集中于尖峰 → 数学上幂级数无法捕获
- **后果**：压缩区间 → 峰值主导 → 泰勒近似 → 高斯积分终结 → **这是整个渐近分析的核心逻辑链**

## 二、积分渐近（Integral Asymptotics）

> y ≈ 260–400 | 渐近根基在积分中的直接应用

### 两大支柱方法

| 方法 | 核函数 | 关键条件 |
|------|--------|----------|
| **Laplace 方法** | $e^{x\phi(t)}$ 实指数 | $\phi(t)$ 在 $t_c$ 取最大值，$\phi'(t_c)=0$ |
| **稳定相方法** | $e^{i\omega\phi(t)}$ 虚指数 | $\phi'(t_c)=0$ 驻点，高频振荡相消 |

### 共同原理
- 指数放大/振荡效应使贡献集中于 **$\phi'(t)=0$ 驻点** 附近
- **关键操作**：驻点局部展开 + 高斯积分终止公式
- **数学内涵一致**：复指数 = 实 + 虚，理论无缝衔接

### 推广：无驻点情形
1. **分部积分法** — 化为有驻点
2. 注：分部积分是后续边界层分析的**必经前置基础**

### 经典例子线索
- Laplace → Gamma 函数 → Stirling 公式
- 稳定相 → Bessel 函数 $J_0(x)$ 的大 $x$ 渐近
- Riemann-Lebesgue 引理：$O(1/x)$ 需要 $C^1$ 条件

## 通用前置基础（Overture to ODEs）

> y ≈ 402–450 | 从积分过渡到 ODE 的三件工具

### 1. 分部积分法
- 储备：局部边界层分析的必经前置基础

### 2. Rescaling（重缩放）
- 引入伸展变量，放大边界层内的物理

### 3. Dominant Balance（主导平衡）
- **用途**：比较各项相对大小 → 确定 $\epsilon$ 的幂指标 → 定位边界层位置
- **应用 1**：判定边界层位置（左边界 / 右边界 / 中间内层 / 拐角层）
- **应用 2**：指定 $\delta = \epsilon^\gamma$，通过主导平衡确定 $\gamma$

## 三、ODE 奇异摄动 → 三部曲

> y ≈ 532–820 | 图中用 ①②③ 串联，逻辑递进

```
Local Breakdown           Global Breakdown           Long-time Breakdown
(空间局部失效)             (全局坍塌)                  (长时间演化失准)
     ↓                         ↓                           ↓
 ① Boundary Layer          ② WKB 方法                 ③ Multiple Scale
```

### ① Boundary Layer（边界层）

**对应**：Local Breakdown — 空间局部失效

**标准五步流程**：
1. **Outer 外解**：$\epsilon \to 0$，服从全局平衡 → 丢失边界条件
2. **判定边界层位置**：主导平衡 → 左/右/中间/拐角
3. **Inner 内解**：引入伸展变换 $X = (x-x_0)/\delta$，专门解析薄层物理
4. **MAE 匹配渐近**：连接内外极限；确定积分常数
5. **复合解**：$y_{\text{unif}} = y_{\text{out}} + y_{\text{in}} - y_{\text{match}}$

**分析框架**：
- 左栏/右栏边界层：$\delta \sim \epsilon$
- 中间内层：$\delta \sim \sqrt{\epsilon}$ 等
- 拐角层：多重尺度交界

### ② WKB 方法（Wentzel–Kramers–Brillouin）

**对应**：Global Breakdown — 全局坍塌
**原因**：$\epsilon^3 y'' + Q(x)y = 0$ 在 $Q(x)$ 变号处失效

**标准流程**：
1. **Ansatz**：$y \sim e^{S_0/\delta + S_1 + \cdots}$
2. **分区划分**：$Q > 0$ 振荡区（三角函数）；$Q < 0$ 实指数衰减区
3. **Turning Point 转折点**：$Q(x_0) = 0$ → WKB 失效
   - 局部尺度 $O(\epsilon^{2/3})$
   - 局部解化为 **Airy 函数**，Airy 函数渐近连接两侧 WKB 解
4. **双转折点**：Bohr-Sommerfeld 量子化条件 → 离散特征值 $\oint \sqrt{Q}\,dx = (n+\frac{1}{2})\pi\epsilon$

**统一性**：WKB 统一处理"线性小参数导数导致的非一致问题"

### ③ Multiple Scale（多尺度分析）

**对应**：Long-time Breakdown — 长时间演化失准

**失效根源**：超长区间上展开，时间积累效应 → 相位累积偏差 → **Secular Term（久期项）**

**对策**：引入多重时间尺度
$$T_0 = t,\quad T_1 = \epsilon t,\quad T_2 = \epsilon^2 t, \ldots$$

**核心目标**：消除久期项（消除共振条件）→ **Amplitude Equation（振幅方程）**

**两种方法对比**：
| 方法 | 适用范围 | 做法 |
|------|----------|------|
| **Poincaré-Lindstedt (PL)** | 仅周期解 | 展开频率 $\omega = \omega_0 + \epsilon\omega_1 + \cdots$ |
| **多尺度分析** | 周期 + 非周期 | 多重时间变量，偏导展开 |

**经典模型**：
- **Van der Pol**：$\ddot{x} + x = \epsilon(1-x^2)\dot{x}$（自激振子 → 极限环）
- **Duffing**：$\ddot{x} + x + \epsilon x^3 = 0$（非线性弹簧）

**拓展**：
- 非自治系统：Rayleigh-Lorentz
- 参数共振：Mathieu 方程

## 四、动力系统与分岔

> y ≈ 900–1150 | 从"解的具体形式"转向"解的定性行为"

### Poincaré-Lindstedt 与分岔的桥梁
- 前置：拓扑简化法 → 化为标准 **Normal Form（规范型）**

### 经典局部范式
| 分岔类型 | Normal Form | 关键特征 |
|----------|-------------|----------|
| **鞍结分岔** Saddle-node | $\dot{x} = r + x^2$ | 两个不动点碰撞消失 |
| **跨临界分岔** Transcritical | $\dot{x} = rx - x^2$ | 不动点交换稳定性 |
| **叉式分岔** Pitchfork | $\dot{x} = rx - x^3$（超临界）/ $\dot{x} = rx + x^3$（亚临界）| 对称性 → 一个变三个 |

## 五、一维分岔（One-dimensional Bifurcation）

> y ≈ 1040–1150 | 分岔分析的标准操作流程

### 核心逻辑闭环（四步）

1. **选取 Bifurcation Parameter** — 关键控制参数 $r$
2. **寻找 Fixed Point 不动点** — $f(x^*, r) = 0$
3. **判定 Stability 稳定性** — 根据分岔点附近 $f'(x^*)$ 符号判断
4. **绘制 Bifurcation Diagram 分岔全景图** — $x^*$ vs $r$

**延伸**：含 §10, §20, Hopf 分岔初步（二维系统 → 极限环）

## 复习线索：抓住"三个失效 → 三种方法"

| 失效类型 | 表现 | 解决方法 | 核心工具 |
|----------|------|----------|----------|
| **Local Breakdown** | 最高阶导数丢失 → 边界条件不满足 | ① Boundary Layer | 伸展变换 + MAE 匹配 |
| **Global Breakdown** | $Q(x)$ 变号 → 振荡↔衰减切换 | ② WKB | Ansatz + Airy 连接 |
| **Long-time Breakdown** | 久期项积累 → 振幅/相位漂移 | ③ Multiple Scale | 多时间尺度 + 消久期项 |

三条链最终汇聚到 **Normal Form + Bifurcation**，完成从"解析求解"到"定性分析"的范式转换。
