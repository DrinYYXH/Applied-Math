# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

2026 春季"高等应用数学"个人学习仓库，所有计算基于 **Wolfram Mathematica 14.3**（`.nb` notebook 文件）。

## 仓库结构

```
HW1/        # 第一次作业：函数展开与反函数级数
HW2/        # 第二次作业：ODE 边界层分析、匹配渐近展开
Final/      # 期末专题（未跟踪）：完整课程笔记 + 专题大纲
```

作业目录内包含题目 PDF (`HW*.pdf`)、提交答案 PDF (`hw*_*.pdf`)，以及 Mathematica 计算笔记本 (`*.nb`)。

## 课程主题

渐进展开、拉普拉斯方法、稳定相方法、Bessel 函数、ODE 局部分析（Liouville-Green）、摄动方法（正则/奇异）、边界层理论（匹配渐近展开）、MKB 方法/Schrödinger 方程、分岔理论（鞍结/跨临界/叉形分岔）、多尺度分析、van der Pol/Duffing 振子、Poincaré-Lindstedt 方法。
